*   [在 ***拉斯边喘息***上同时安装 Python 2.7 和 Python 3+的 OpenCV 3.0。](https://pyimagesearch.com/2015/07/27/installing-opencv-3-0-for-both-python-2-7-and-python-3-on-your-raspberry-pi-2/)

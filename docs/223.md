在本教程中，我们将使用 [picamera](https://picamera.readthedocs.org/en/release-1.9/) ，它为相机模块提供了一个纯 Python 接口。最棒的是，我将向您展示如何使用 picamera 来捕捉 OpenCV 格式的图像。

这个例子将运行在 **Python 2.7/Python 3.4+** 和 **OpenCV 2.4.X/OpenCV 3.0+** 上。
